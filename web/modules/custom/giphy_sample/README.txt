This module provides integration for Giphy.com This will allow plain text fields 
to be displayed as a Gif based on the keyword entered on the plain text field.

## How to use
  1. Go to /admin/structure/types/manage/<any content type>/display.
  2. Select any text field and click on the dropdown option under FORMAT section.
  3. Select "Giphy Field Format" as display format and save.
